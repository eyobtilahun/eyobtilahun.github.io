Meda Chat Desktop Version Concept Design

UI/UX Concept Design

Meda Chat is a platform for students, business men & women, financial institution, government offices and center for a plethora of things including event ticket sales, air time sales, DSTv payment, game… Meda Chat aims to revolutionize messaging & money in Ethiopia the same way WeChat disrupted China.

I designed the Concept User Interface and User Experience by Considering if Meda Chat Android has a Desktop Version Without losing the same User interface component and User Experience. i also added the Dark Mode version of Design Concept for Meda Desktop.
